# getdbmode

Get database mode.

## Request Json Syntax

| **Key** | **Description** |
| --- | --- |
| task | task name |
| token | token string encrypted. |
| dblist | database name |

## Request Sample

```
{
  "task": "getdbmode",
  "token": "cdfb4c5717170c5e6f12e5b1643a2b67132bcc7d82bd6090e92a55cddd5950db7926f07dd201b6aa",
  "dblist": "demodb"
}
```
